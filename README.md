# contas-vuejs

## Installation

Install npm dependencies

    sudo npm install -g lite-server
    sudo npm install -g concurrently
    sudo npm install -g node-sass

    npm install

Install composer dependencies

    cd api
    composer install

## Run Api

    php -S 127.0.0.1:8000

## Run FrontEnd + Babel compile

    npm start

## Run FrontEnd

    lite-server

## Run Babel compile

    npm run babel
